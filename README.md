# Royal Bites - Restaurant Web App

I built this project to learn full stack web development. 
It started as a simple HTML website and I gradually added 
a Python Flask backend, SQLite database, and admin panel.

## What I Built

- Restaurant menu that loads from a database
- Contact form that saves messages to database  
- Admin panel where I can add and delete menu items
- Mobile responsive design

## Tech I Used

- Python Flask for the backend
- SQLite to store menu items and messages
- SQLAlchemy to connect Python with the database
- Jinja2 to show database data in HTML
- HTML, CSS, JavaScript for the frontend

## How to Run This Project

1. Install Flask
pip install flask flask-sqlalchemy

2. Create the database
python app.py
then visit http://127.0.0.1:5000/setup

3. Add menu items
python seed.py

4. Start the app
python app.py

Go to http://127.0.0.1:5000

## Admin Panel
Go to http://127.0.0.1:5000/admin to add or delete menu items
